[[MCA]] will refer to the main character
Characters will be referred to as codes instead of their names
Use this website https://www.random.org/strings/
Format will be (Prefix) + (4 digit A-Z)

Codes will be in the following format

| Type | Prefix |
| ---- | ---- |
| Characters | C |
| Items | I |
| Locations | L |
Example:
[[CJBWZ]] will now refer to Simon

But to make things easier when script writing, Use CTRL + P and then search "Replace"
Type out names in ALLCAPS as they are but then replace them in post

example: SIMON
